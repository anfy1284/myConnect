# MyConnect

MyConnect - это система для перехвата и туннелирования сетевого трафика через центральный сервер с использованием WinDivert.

## Структура проекта

Проект состоит из двух основных компонентов:

*   **[Server](server/README.md)**: Центральный сервер на Python (WebSocket), который маршрутизирует трафик между клиентами. Разворачивается на облачном сервере (Linux/Windows).
*   **[Client](client/README.md)**: Клиентское приложение для Windows. Использует драйвер WinDivert для перехвата пакетов и перенаправляет их через сервер. Работает в системном трее.

## Основные возможности

*   **Перехват трафика**: Гибкая фильтрация пакетов (TCP/UDP, IP, порты) с помощью WinDivert.
*   **Туннелирование**: Передача перехваченных пакетов через WebSocket (поддержка TLS/WSS).
*   **Маршрутизация**: Возможность перенаправлять трафик от одного клиента к другому (например, использовать домашний ПК как выходную ноду для рабочего ноутбука).
*   **Управление**: Клиент работает в фоне с иконкой в трее для быстрого управления.

## Быстрый старт

1.  **Настройте сервер**:
    *   Отредактируйте `server/config.json`, добавив допустимых клиентов.
    *   Запустите сервер.
2.  **Настройте клиент**:
    *   Скачайте драйвер WinDivert.
    *   Отредактируйте `client/config.json`, указав адрес сервера и фильтры перехвата.
    *   Запустите клиент (требуются права администратора).

Подробные инструкции см. в папках компонентов.
